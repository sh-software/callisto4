    star-rating

    @mixin star-rating($default-color: $gray-lighter, $selected-color: #FFD700, $hover-color: #FFED85) { ... }

    Description

    Element group for star rating

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $default-color

    Normal color of the star. Defaults to the Bootstrap var $gray-lighter.

    		Color$gray-lighter
    $selected-color

    Selected color of the star. Defaults to #FFD700.

    		Color#FFD700
    $hover-color

    Hover color of the star. Defaults to #FFED85.

    		Color#FFED85

    Example
